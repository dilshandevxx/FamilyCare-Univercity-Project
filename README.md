### 🏁 ✨ Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/dilshandevxx/FamilyCare-Univercity-Project.git
   ```
2. **Setup Backend**
   ```bash
   cd backend && npm install && cp .env.example .env
   ```
3. **Setup Frontend**
   ```bash
   cd frontend && npm install && npm run dev
   ```
